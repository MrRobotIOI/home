Project 3

Java / solo project

Uses a graph along with stack, queue and vertex classes I made. Solves for the number of steps needed to get to a specific floor given the values of N (number of floors), U (up by x), D(down by x), X (starting floor), and Y (destination floor). Returns the answers within 1 second.
Imagine there is a building with N floors. And you need to get to a particular floor from whatever floor you’re currently on using an elevator. The elevator only has 3 buttons.
UP: a button saying “Up by U" where U is a positive integer (for example, \Up by 3"). By pushing this button, you will go up U floors, or until you reach the top floor.
Down: a button saying “Down by D" where D is a positive integer (for example, \Down by 21"). By pushing this button, you will go down D floors, or until you reach the ground floor (the 1st floor)
Jump: button saying “JUMP 2x", which will bring you to the floor number that is twice of the current one or hit the top floor. For example, if you are currently on floor 12, then pushing the JUMP button will bring you to floor 24, or to the top floor if the top floor is less than 24.
